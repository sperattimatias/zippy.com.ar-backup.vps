# Visual Identity Zippy — Ultra Premium Black Edition

## Palette
- Background: `#0F0F12`
- Surface: `#1F1F24`
- Divider: `#2A2A30`
- Primary accent: `#2ED573`
- Text secondary: `#B8B8C0`

## Principles
1. Sobriedad: evitar blancos puros y saturación excesiva.
2. Claridad: jerarquía visual con espacio negativo y tipografía simple.
3. Motion elegante: transiciones breves, suaves, sin rebotes bruscos.
4. Consistencia: todos los sheets/cards/buttons comparten radios, spacing y contraste.

## UI Guidelines
- Padding horizontal estándar: 20.
- Cards: fondo `surface`, borde divisor suave, sombra baja.
- Chips de estado: pill oscura con punto activo verde animado.
- Empty states: iconografía lineal y CTA outline.
